A component library for developing [Tuya](https://www.tuya.com) panels.

## Get Started

### Install

```shell
$ npm install tuya-panel-kit

# or

$ yarn add tuya-panel-kit
```

### Usage

```tsx
import { Button } from 'tuya-panel-kit';

<Button />;
```
